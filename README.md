# 📝 Todo List

A modern and responsive To-Do List web application built using HTML, CSS, and JavaScript. This application allows users to efficiently organize daily tasks by adding, editing, deleting, searching, filtering, and prioritizing them through a clean and user-friendly interface.

---

## 🚀 Features

- Add new tasks
- Edit existing tasks
- Delete tasks
- Mark tasks as completed
- Set task priority (High, Medium, Low)
- Search tasks instantly
- Filter tasks by status and priority
- Sort tasks
- Responsive design for different screen sizes
- Modern and attractive user interface

---

## 🛠 Technologies Used

- HTML5
- CSS3
- JavaScript (Vanilla)

## ▶️ How to Run

1. Download or clone this repository.
2. Open the project folder.
3. Double-click the `index.html` file or open it in any web browser.

No installation or additional software is required.
